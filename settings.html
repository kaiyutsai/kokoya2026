這個頁面只有管理員（凱宇）能進入。

回主控台

⚙️ 系統設定

這裡管理共用下拉清單，員工輸入時可以從這裡的選項挑

🏪 供應商

在「進貨/庫存」開立進貨單時，供應商欄位會出現這裡的選項。仍可手動輸入新名稱。

🚚 配送員

在「訂單批次」建批次時，配送員欄位會出現這裡的選項。

🔑 API 金鑰

Gemini AI 解析訂單用的金鑰。存在 Firestore 加密保護，只有登入後的家人能讀取，不會出現在公開的 GitHub repo。
到 Google AI Studio 申請新金鑰，貼到下面儲存即可。

👤 經辦人

在「銷貨／進貨／出貨單」表單的「經辦人」下拉選單會出現這裡的選項。
預設帶 4 位家人，可加入臨時幫手（例如阿嬤週末來幫忙）。

📊 系統資料總覽

點上方「🔄 重新載入」開始查詢…

⚠️ 危險區

以下操作會永久刪除對應的全部資料，無法復原。每個動作都會要求你輸入特定文字確認。
通常只在「測試完成想清空試打的單」、「重新開帳」這類情境用。
