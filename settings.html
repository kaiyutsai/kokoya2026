⚙️ 系統設定

這裡管理共用下拉清單，員工輸入時可以從這裡的選項挑

🏪 供應商

在「進貨/庫存」開立進貨單時，供應商欄位會出現這裡的選項。仍可手動輸入新名稱。

🚚 配送員

在「訂單批次」建批次時，配送員欄位會出現這裡的選項。

🔑 API 金鑰

Gemini AI 解析訂單用的金鑰。存在 Firestore 加密保護，只有登入後的家人能讀取，不會出現在公開的 GitHub repo。
到 Google AI Studio 申請新金鑰，貼到下面儲存即可。

💳 銀行帳號（線上轉帳）

客人在購物網結帳時會看到這個帳號。設定後購物網會自動同步顯示。
提醒：這份資料是公開可讀的（任何訪客都能看到），請只放收款用的帳號。

📦 運送方式設定

客人在購物網結帳時可選 3 種：自取 / 冷藏低溫配 / 老闆親送。
以下設定會即時同步到結帳頁。

📍 自取地點

客人選「自取」時看到的地點清單，可填多個。建議寫上地址 + 營業時間。

❄️ 冷藏低溫配 — 運送公司

客人選低溫配時顯示的物流商名稱。預設黑貓宅急便。

📦 冷藏宅配 箱型 + 運費（系統會依購物數量自動估）

以「蘋果顆」為換算單位（1 蘋果 ≈ 1 單位）。客人下單時，系統會把購物車總數量加總後，挑能裝下的最小箱顯示運費。
超過大箱容量會自動估「大箱 × N 個」。如果是西瓜、鳳梨這種特大水果，建議手動聯絡客人收費。

🛵 老闆親送 — 可達範圍

客人選「老闆親送」時會看到這個地區提示，地址也會自動驗證是否包含這幾個關鍵字。
多個區域用逗號或頓號分隔（例：台南市南區、東區、安平區）。

🍎 水果大項分類

水果大項分類，用於品項清單、報表、銷售統計表的分組。
這裡管理「分類」清單，到「品項清單」選擇分類後，該品項就會歸類到這個大項底下，點分類就能看屬於它的所有品項。

📦 「」分類底下的品項
→ 到品項清單修改分類歸屬

👤 經辦人

在「銷貨／進貨／出貨單」表單的「經辦人」下拉選單會出現這裡的選項。
預設帶 4 位家人，可加入臨時幫手（例如阿嬤週末來幫忙）。

⚠️ 危險區

以下操作會永久刪除對應的全部資料，無法復原。每個動作都會要求你輸入特定文字確認。
通常只在「測試完成想清空試打的單」、「重新開帳」這類情境用。
